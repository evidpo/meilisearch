# Краткое описание проекта

**Название:** Поисковый виджет evidpo.ru  
**Платформа сайта:** Creatium  
**Поисковый движок:** Meilisearch (Railway)  
**Синхронизация:** n8n  
**Стадия:** MVP готов, в эксплуатации

**Суть:** Полноэкранный оверлей поиска курсов для сайта evidpo.ru. Мгновенный поиск по 200+ курсам с автодополнением, динамическими категориями и популярными запросами. Данные синхронизируются из Directual через n8n workflow.

**Ключевые метрики:**
- Время ответа: ~5ms
- Курсов в индексе: 201
- Категорий: 10 (топ по студентам)
